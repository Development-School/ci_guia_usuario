Instruções de Instalação

O CodeIgniter é instalado em quatro passos:

  1. Descompacte o pacote.
  2. Envie as pastas e arquivos do CodeIgniter para seu servidor. Normalmente, o arquivo index.php estará em sua raiz.
  3. Abra o arquivo application/config/config.php com um editor de texto e defina sua URL base. Se você pretende usar a criptografia ou sessões, definir a sua chave de criptografia.
  4. Se você pretende usar um banco de dados, abra o arquivo application/config/database.php em um editor de texto e altere as configurações do seu banco.

Se você deseja aumentar a segurança, escondendo a localização de seus arquivos no CodeIgniter, você pode renomear as pastas system e application para maior privacidade. Se você renomeá-los, você deve abrir seu arquivo index.php e definir as variáveis $system_folder e $application_folder, que ficam no início do arquivo, para o novo nome que escolheu.

Para a melhor segurança, a pasta system e a pasta application devem ser colocadas acima do diretório raiz para que eles não sejam acessíveis directamente através de um navegador. Por padrão, arquivos .htaccess estão incluídos em cada pasta para ajudar a impedir o acesso direto, porém movê-los é melhor para removê-los do total acesso ao público no caso das configurações do servidor web não respeitarem o .htaccess.

Depois de movê-los, abra o arquivo index.php e coloque nas variaveis $system_folder e $application_folder, o caminho completo dos diretórios, Ex. '/www/MyUser/system'.

Uma medida adicional pode ser tomada em ambientes de produção é desabilitar o relatório de erros do PHP e qualquer outra funcionalidade para ambientes de desenvolvimento. No CodeIgniter, isto pode ser feito definindo a constante ENVIRONMENT nas configurações, essa constante é descrita detalhadamente na página de segurança.

Só isso!

Se você acabou de conhecer o CodeIgniter, por favor leia a seção Primeiros Passos do Guia do Usuário para começar aprender a construir aplicações PHP dinâmicas. Divirta-se!
