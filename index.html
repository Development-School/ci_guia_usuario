Bem vindo ao CodeIgniter

CodeIgniter é um Framework de Desenvolvimento de Aplicações - um conjunto de ferramentas - para quem constrói sites usando PHP. Seu objetivo é que você possa desenvolver projetos muito mais rápido do que você poderia se estivesse escrevendo código a partir do zero, fornecendo um rico conjunto de bibliotecas para as tarefas mais comuns, bem como uma interface simples e estrutura lógica para acessar essas bibliotecas. CodeIgniter permite que você mantenha o foco em seu projeto minimizando a quantidade de código necessário para uma dada tarefa.

Para Quem é o CodeIgniter?

CodeIgniter é a melhor escolha para você se:

  • Você quer um framework com uma pegada leve.
  • Você precisa de performace excepcional.
  • Você precisa de ampla compatibilidade entre as versões do PHP e diferentes configurações de servidores.
  • Você quer um framework que requer algo proximo a nem uma configuração.
  • Você que um framework que não requer o uso de linha de comando.
  • Você que um framework que não requer aderir a regras restritiva de códigos.
  • Você não quer ser forçado a aprender uma linguagem de templates ou temas (embora um interpretador de templates está disponível opcionalmente, se você precisar de um).
  • Você evita a complexidade, favorecendo soluções simples.
  • Você precisa ser claro, na documentação inteira.
