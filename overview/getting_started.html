Primeiros Passos no CodeIgniter

Qualquer software exige algum esforço para aprender. Nós fizemos o nosso melhor para minimizar a curva de aprendizagem, tornando o processo mais agradável possível.

O primeiro passo é instalar o CodeIgniter, em seguida, ler todos os tópicos na seção Introdução do Índice.

Em seguida, ler cada uma das páginas dos Tópicos Gerais em ordem. Cada tópico baseia-se no anterior, e inclui exemplos de código que nos incentivamos a tentar fazê-los.

Uma vez que você compreender os conceitos básicos você estará pronto para explorar as páginas das Referência das Classes e as Referência dos Helpers para aprender a utilizar as bibliotecas nativas e arquivos auxiliares.

Sinta-se livre para tirar proveito de nosso Fórum da Comunidade, se você tiver dúvidas ou problemas, entre no nosso Wiki para ver exemplos de código postadas por outros usuários.
