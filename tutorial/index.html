Tutorial − Introdução

Este tutorial destina-se a apresentá-lo ao framework CodeIgniter e aos princípios básicos da arquitetura MVC. Ele irá mostrar a você como uma aplicação básica é construída passo-a-passo com o CodeIgniter.

Neste tutorial, você estará criando uma aplicação básica de notícias. Você vai começar escrevendo um código para carregar páginas estáticas. Em seguida, você irá criar uma seção de notícias que lê as notícias a partir de um banco de dados. Por último, você vai criar um formulário para adicionar novas notícias no banco de dados.

Este tutorial irá focar-se essencialmente em:

  • Princípios básicos do MVC
  • Princípios básicos do roteamento
  • Validação de formulário
  • Executar consultas básicas em banco de dados usando "Active Record"

Todo o tutorial é dividido em várias páginas, cada uma explicando uma pequena parte das funcionalidades do framework CodeIgniter. Você vai passar pelas páginas a seguir:

  • Introdução, esta página aqui, traz uma visão geral do que vem pela frente.
  • Páginas Estáticas, que vai te ensinar o básico de controllers, views e routeamento.
  • Seção de Notícias, onde você vai começar a usar models e vai fazer algumas operações básicas em banco de dados.
  • Criar Notícias, que irá introduzir operações de banco de dados mais avançadas e validação de formulário.
  • Conclusão, o que lhe dará algumas indicações sobre leitura adicional e outros recursos.

Aproveite a sua exploração no framework CodeIgniter.
